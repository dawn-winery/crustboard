# crustboard

Starboard replacement

### COMMANDS
- `/addboard <name> <dest-channel> <reactions> <min_reactions>?`
- `/showboard <name>?`
- `/editboard <name> <dest-channel>? <reactions>? <min_reactions>?`
- `/deleteboard <name>`
- `/leaderboard <name>?`
- `/moststarred <name>?`
- `/random <name>?`
